API documentation

Get user by idName

GET v1/api/user/{idName}

Example

http://localhost:8080/v1/api/user/103277733

Response

{ "id" : 16, "idName" : "103277733", "phoneNumber" : "27716125197", "status" : "FIXED", "message" : "add prefix 27" }
